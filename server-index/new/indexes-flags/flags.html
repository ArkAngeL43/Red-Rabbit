0x102 - Interacting with red rabbit using flags
Section 0x102 will be talking about flags and their usages red rabbit relies heavily on flags and user input given its natural ability to be a framewok.
> --filepath --filepath is a flag that can be used to specify a filepath, its usage is as follows sudo ./main --filepath="examples/pcap/ospf.pcap"
> --wordl, --hashl, --userl all of these flags are similar, thse flags stand for hash, user, and word lists to be used for things like brute forcing attacks, hash attacks, and other attacks that may need a wordlist. its usage is quite simple --wordl if you want to specify a wordlist of passwords --userl if you want to specify a wordlist of username --hashl if you want to specify a wordlist of hashes they are quite simple to use
> -i, --input, -o, --output This is used for most stegonography utilities -i or --input can be used as a input image -o or --output can be used as an output image or output filename This flag isnt used much in the script however it can still come in handy espcially when it comes to commands that rely on a input and output filename, most of this is just due to how the script was built around certian modules
> --CIDR, -z -z and --CIDR are commands that are used to run the CIDR of a ping command outside of ping arp see section 0x010+ for more information or the ping modul section for more information --CIDR can be used as the following --CIDR="10.0.0.1/24"
> --payload --payload is used to specify a payload to either inject into images, search for in payload parsing, parse a payload, or really any command that requires a payload an example of using this flag is --payload="script>alert()script>"
> --interface the --interface flag is used to specify a wireless interface to use for network sniffing or network based attacks that may need a interface for purposes of specification example --interface="wlan0mon"
> --passlen --passlenis a flag to set the length of a string used for password / wordlist generation. The flag takes a number value and does not have a limit of how long each generation string can be. the defualt length for this is 16 example --passlen="16"
> --type the --type flag allows you to specify a certian chunk of a PNG image before you inject payloads into it, an example usage is of the following --type="IEND"
> --offset specify a offset to inject images at most of this helps with PNG image injection and is required before you inject PNG formats an example usage is --offset="0x00001"
> --reso This is a unqiue flag that was added for user customization, basically with three options this allows you to customize the way files will be outputted or data will be formatted all three options -> landscape, verticle, none, shark none will not give you any banner and will just start the console landscape will output a landscape like banner any of the few shark will output a larger shark like banner with larger less veritcle will output a simple banner with simple output usage: --reso="options>"
